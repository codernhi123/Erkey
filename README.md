# Nomo
## This is a personal **Computer Vision & System Programming** project from Fintanyl.

* The repo comes in with a prototype (Python) and a final product (C++) directory. Everything will be made in python and later converted into C++ for optimization.

## Expected outcome: 
The project aims to create an alternative means of controlling cursor rather than the conventional physical mouse.

Inspired from the keyboard high-utility of NeoVim, the software uses openCV with mediapipe (facemesh) to track, seperate
and stabilize the movement of the pupils' average center to project onto the monitor for cursor manuevering.

For practical uses, the "Edit" mode can be enabled using a combination of hotkeys to allow the cursor's movement.
